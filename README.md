## workflow

1. fire up [Juno](https://junolab.org/)
2. copy [tmpl.jl](./tmpl.jl) and paste and save it into a file (let to be `prob.jl`)
3. create a sample input file and name it `prob.in`
4. run `prob.jl` interactively and solve !

> [tmpl.jl](./tmpl.jl)
